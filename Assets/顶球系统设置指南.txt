========================================
        顶球杂技辅助系统设置指南
========================================

📁 文件说明
----------------------------------------
- BallCatcher.cs        - 2D版本（可选）
- BallCatcher3D.cs      - 3D版本（推荐！）
- HeadBallSpawner.cs    - 球生成器

🔧 完整设置步骤
========================================

1️⃣ 设置角色的BallCatcher系统
----------------------------------------
1. 在场景中找到：Meshy_AI_Black_Robed_Wushu_War_biped_Character_output
2. 找到它的子物体：BallCatcher
3. 在BallCatcher的子物体中选择：HeadTrigger
4. 添加组件：BallCatcher3D
5. 在Inspector中设置：
   - Front Guard  = 拖入 FrontGuard
   - Back Guard   = 拖入 BackGuard
   - Left Guard   = 拖入 LeftGuard
   - Right Guard  = 拖入 RightGuard

2️⃣ 确保碰撞体设置正确
----------------------------------------
HeadTrigger:
- BoxCollider (3D版本!)
- Is Trigger = true
- 调整大小：覆盖头顶区域

四个挡板 (Front/Back/Left/Right Guard):
- BoxCollider (3D版本!)
- Is Trigger = false
- 不需要 Rigidbody
- 不需要 SpriteRenderer（保持隐形）

3️⃣ 设置球生成器
----------------------------------------
1. 在场景中创建空物体，命名为：BallSpawner
2. 添加组件：HeadBallSpawner
3. 在Inspector中设置：
   - Ball Prefab  = 拖入你的球预制体
   - Head         = 拖入角色的头部或BallCatcher
   - Height       = 1.5（根据需要调整）

4️⃣ 设置球预制体
----------------------------------------
确保球预制体有：
1. SphereCollider (3D)
2. Rigidbody (3D, Use Gravity = true)
3. Tag = "Ball" (不需要手动设置，生成器会自动设置)

🎮 测试和使用
========================================

按键控制：
- Space  = 生成球
- R      = 重置球
- X      = 销毁球

功能说明：
- 球落下，进入HeadTrigger → 挡板自动关闭
- 球离开HeadTrigger → 挡板自动打开
- 平滑旋转动画
- 可调整参数：openAngle, closedAngle, rotationSpeed

🔧 参数调整
========================================

在BallCatcher3D中调整：
- openAngle: 45 (挡板打开角度，太大/太小试试)
- closedAngle: 0 (关闭角度)
- rotationSpeed: 8 (旋转速度，越大越快)
- ballTag: "Ball" (球的标签，不需要改)

在HeadBallSpawner中调整：
- height: 1.5 (生成高度，球离头顶多高)
- showDebug: true (显示调试信息)

📝 常见问题
========================================

Q: 挡板不转？
A: 检查是否是3D版本！用BallCatcher3D而不是BallCatcher

Q: 球不触发？
A: 检查球的Tag是否是"Ball"，碰撞体是否正确

Q: 挡板方向反了？
A: 在OpenGuards()里调整角度符号（正负号）

Q: 看不到挡板？
A: 挡板是隐形的，正常！可以添加SpriteRenderer调试
